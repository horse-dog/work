## 不用加减乘除做加法

--------------------

- 写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

--------------------

- 示例:

    - 输入: a = 1, b = 1

    - 输出: 2

--------------------

- 限制：

    - a, b 均可能是负数或 0

    - 结果不会溢出 32 位整数

--------------------

- 加法器的实现原理

```cpp
class Solution {
public:
    int add(int a, int b) {
        while (b != 0) {
            unsigned int carry = (unsigned int)(a & b) << 1;
            a = a ^ b;
            b = carry;
        }
        return a;
    }
};
```
